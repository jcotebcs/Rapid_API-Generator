401 Unauthorized Error

Causes:

  • Invalid or missing Notion API key
  • API key format is incorrect
  • Integration has been disabled

Solutions:

  • Check your API key at Notion Integrations
  • Ensure the key starts with "secret_"
  • Regenerate the integration if necessary
403 Forbidden Error

Causes:

  • Integration not shared with the database
  • Insufficient permissions
  • Database doesn't exist or is private

Solutions:

  • Go to your Notion database ‚Üí Share ‚Üí Invite your integration
  • Grant "Edit" permissions to the integration
  • Verify the database ID is correct
404 Not Found Error

Causes:

  • Incorrect database ID format
  • Database ID doesn't exist
  • Integration doesn't have access

Solutions:

  • Extract database ID from Notion URL correctly
  • Ensure database is shared with integration
  • Check for typos in the database ID
CORS Issues

Problem: Browser blocks direct API calls to Notion

Solution: Use a backend proxy server

  • Deploy the included notion_api_backend.js
  • Set backend URL in configuration above
  • Never expose API keys in frontend code
